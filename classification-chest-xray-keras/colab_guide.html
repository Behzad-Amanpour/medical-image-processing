// 01 — Overview

What is Google Colab?

Google Colaboratory (Colab) is a free, cloud-based Jupyter notebook environment that runs entirely in your browser. It requires no setup, gives you free access to GPUs, and lets you write and execute Python code in real time — perfect for data science, machine learning, and research workflows.

All you need is a Google account. Your notebooks are saved directly to your Google Drive.

// 02 — Setup

Setting up your notebook

Follow these steps to create and configure your first Google Colab notebook:

1
Go to Google Colab

Visit colab.research.google.com and sign in with your Google account.

2
Create a new notebook

Click File → New notebook. A new .ipynb file will be created and saved to your Drive.

3
Change the runtime type (optional)

Go to Runtime → Change runtime type and select GPU if you plan to train machine learning models.

4
Write your first cell

Click inside a code cell, type your Python code, then press Shift + Enter to run it.

5
Install packages if needed

Use !pip install package_name directly in a cell to install any Python library.

// 03 — Google Drive

Mounting Google Drive

By default, files you create in Colab are temporary and lost when the session ends. Mounting your Google Drive gives your notebook permanent access to your files — datasets, saved models, outputs — so nothing is lost between sessions.

How mounting works

📓
Colab Notebook
Your runtime session
mount()
📁
/content/drive
Local mount point
syncs to
☁️
Google Drive
Your cloud storage

Paste the following code into a cell and run it. You will be prompted to authorize access to your Google Drive — click the link, sign in, and copy the authorization code back into the field.

Python 
# Step 1: Import the drive module from Google Colab
from google.colab import drive

# Step 2: Mount Google Drive at /content/drive
drive.mount('/content/drive')

# Step 3: Verify it's mounted by listing Drive contents
import os

drive_path = '/content/drive/MyDrive'
print("Files in your Google Drive:")
for item in os.listdir(drive_path):
    print(f"  📄 {item}")
✅

After running, you will see Mounted at /content/drive as the output. Your entire Google Drive is now accessible from the path /content/drive/MyDrive/.

Once mounted, you can read and write files just like any local path:

Python — Reading & Writing files 
import pandas as pd

# Read a CSV file from your Drive
df = pd.read_csv('/content/drive/MyDrive/your_folder/data.csv')
print(df.head())

# Save a file back to your Drive
df.to_csv('/content/drive/MyDrive/your_folder/output.csv', index=False)
print("File saved to Google Drive successfully!")
⚠️

Note: You need to mount Drive every time you start a new Colab session. The mount does not persist between sessions, but your files on Drive remain safe.
